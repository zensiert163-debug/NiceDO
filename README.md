# NiceDO - Space MMO Game

Ein DarkOrbit-ähnliches Weltraum-MMO Spiel.

## Features

- Raumschiff-Verwaltung
- PvP-Kämpfe
- Ressourcen-Mining
- Quest-System
- Gilden/Clans

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js/Express (optional)
- Grafiken: Canvas/WebGL

## Installation

```bash
git clone https://github.com/zensiert163-debug/NiceDO.git
cd NiceDO
npm install
npm start
```
